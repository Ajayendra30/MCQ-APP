# MCQ QuizSphere - Online Quiz Web Application

## 📌 Project Overview

MCQ QuizSphere is a web-based quiz application developed using Java technologies.
The system allows students to take multiple-choice quizzes and view their scores.
It also provides an admin panel to manage questions and users.

This project demonstrates the use of Java web technologies such as JSP, Servlets, and JDBC.

---

## 🚀 Features

* Student registration and login
* Attempt multiple-choice quizzes
* Automatic score calculation
* Admin panel to manage quiz questions
* User-friendly interface
* Secure login system

---

## 🛠️ Technologies Used

* Java
* JSP (Java Server Pages)
* Servlets
* JDBC
* HTML
* CSS
* Oracle Database
* Apache Tomcat Server

## ⚙️ How to Run the Project

1. Install **Java JDK**
2. Install **Apache Tomcat Server**
3. Install **0racle Database**
4. Import the project into **Eclipse IDE**
5. Configure database connection
6. Run the project on Tomcat server
